## ✨ 功能
- Material Design 3风格
- 支持深浅模式切换
- 适配平板
- 支持章节缓存
- 支持查看评论与回复
- 阅读进度保存

  ```
  [√] Flutter (Channel stable, 3.38.7, on Microsoft Windows [版本 10.0.26200.7840], locale zh-CN)
  ```
